Windows: Managing two python versions in Anaconda

data science, Python

Hello Everyone,

The idea of this post came across while I was struggling to manage two versions of python in Anaconda.
As a Python novice, I started using Anaconda -Ipython Notebook and Spyder IDE.
I really liked the user-friendliness of these environments. I was using Python 3.5 in Anaconda.

The problem started when I was trying to learn some concepts and projects in python. Those included some libraries which are only supported by Python 2.7 but not Python 3.5
I did not want to uninstall Anaconda and re-install an older version which supports Python 2.7 because there are many other projects for which Python 3.5 is required.

 

If you are also struggling with similar problem of installing two different versions of Python in Anaconda, then you can follow these steps –

1. Ensure that Anaconda path is set in your system environment variables. If not, then set PATH environment variable –

 

2. Run CMD-command prompt as Administrator and try out conda command as –

conda –version

It will successfully run and result the version of conda installed.

 

3. Now, create a new Python 2.7 environment in conda with the following command –

conda create -n py27 python=2.7 anaconda

In this command, py27 is the name of the new environment being created. You can change the name as per your need.
This will take some time and create a new Python 2.7 version.

 

4. After it created, you can activate new environment by entering following command –

activate py27

 

5. After new Python 2.7 version got activated, you can start Anaconda Navigator where you can choose which environment to choose. ‘root’ is the default python environment set in Anaconda which would be Python 3.5 in this case. Here is the screenshot to select the Python environment in Anaconda-

 

6. After selecting Python 2.7 environment in Anaconda Navigator, you can Launch Ipython Notebook or Spyder IDE as per your requirement and verify the version of python in that. It will be Python 2.7

 

Thanks for following the tutorial. 🙂

Random Questions

data science

Hi Everyone,
You can find various random questions related to data science field on this page –

 

Q1 – What if data in training and test dataset is very much different? (Then, the model which we will make on training dataset will not be applicable to test dataset)
Ans – If training and test dataset are very much different then they are not satisfying the general rule of statistics that both training and test dataset should be IID – Independent and Identically Distributed. This means that all the data which is collected(training and test) is taken from the same source, and drawn from the same distribution.

 

Q2 – What is Bias and Variance?
Ans – Bias means, ‘how much on an average are the predicted values different from the actual value’ while

Variance means, ‘how different will the predictions of the model be at the same point if different samples are taken from the same population’.
